作为一个MC 服务器，成员之间的交流是不可或缺的，当然，我们也有义务维护RIA的良好社交氛围，这一章讲述了在RIA进行社交所需要遵守的一些规则和社群统一规定的红线与黑线，接下来我将会对部分规则进行讲解；

## 红黑守则

红黑守则是社区禁止的和鼓励的主要事项， 是进入RIA所有服务器需要遵守的第一守则，包括：

### 红线 （禁止事项）
所有成员必须遵守的条例
1. 未经允许，不得擅自拿取或借用他人容器中的物品，即使它们未上锁（访册除外）；
这是最重要的红线，每一个RIA玩家都应该自觉遵守，不偷拿他人财物和藏品；
2. 游戏内没有“领地”功能，但未经允许绝不能私自拆毁、挪动他人的建筑、设施以及交通线路等成果；
地图上的部分设施属于共有财产，由各个社团和运营社一起运营，破坏这些设施被视为影响其他玩家游戏体验，将会受到惩罚；
3. 严禁通过第三方软件做出超出正常玩家能力、严重影响平衡的行为，详情请阅读RIA Wiki的第三方软件使用引导；
请所有玩家自觉维护自己绿色玩家的身份，通过任何超出自己能力所得的都会被严肃处理；
4. 严禁在 RIA 社群与 RIA 服务器内进行任何违法犯罪行为。发现相关行为应立即向运营社举报。
或者联系你能联系上的任何一个社群管理员也可以，他们大多都是运营社的成员；

### 黑线 （倡议事项）
倡议所有成员所遵守但是并非强制的条例
1. 砍树应将树干砍伐完整；谨慎大幅削平地形，建造突兀的建筑设施；
虽然奇观很好看，但是依势而建的建筑往往更加美观，不是吗？
2. 机械 / 设施应设计具有停止功能，并在停用时及时关闭。
请理解RIA作为公益服务器，难以使用极其豪华的硬件设备，随手关掉机器说不定就能让其他成员的游玩体验变好一分；
3. 建设选址前需全面调查用地冲突情况，并清晰标注预定区域，避免引发纠纷；
合理的沟通是必要的，提前规划并标注有助于减少争端，减少运营组的负担；
4. 借用他人机械设施前应先取得许可，并确保自己了解如何使用再操作；
这里还有一条隐藏的规则：如果导致设施损坏，使用者有责任修复机器，恢复其正常的运行；
5. 在服务器高峰期避免使用负载较大的设施，尊重他人游戏体验；
如上文所言，随手关掉机器说不定就能让其他成员的游玩体验变好一分；
6. 友好待人，不在公共场合或私下对其他成员进行恶意谩骂诋毁；
保持一个良好的聊天环境，少一分戾气，多一份理解；
7. 通过早安与晚安，与其他成员更多地打招呼吧；留下声音，利用满地乱插的留言牌，在世界中给他人留言吧。
事实上，绝大部分地标都有专门设置的留言墙，如果没有，随手插在不妨碍他人正常活动的地方也是可以理解的；

### 接待新人行为规范

#### 不能做的：
1. 不向新人提供游戏中后期才能获得的物品（比如：鞘翅）；
2. 不主动加快新人的游戏进度，如给予大量货币、推荐生产机械等；
3. 不能在新人未表态想要加入聚落 / 组织时，主动邀约新人加入己方聚落 / 组织。
#### 推荐做的：
1. 友善地问候并表示欢迎，积极答疑；
2. 介绍 RIA 的社群环境与结构，并帮助其融入；
3. 介绍你所钟爱的服务器游玩方式与旅游线路。

红黑守则是必须遵守的，接下来是我和运营组的一点小建议：

**在社群中与他人交谈时：**

- 尊敬其他成员，请勿揶揄他人；
- 讨论或争论时请勿上升到人身攻击；
- 禁止使用含有攻击、侮辱意味、令人反感的流行句与梗；

**在社群中与他人讨论时：**
- 适当的讨论有助于双方更加深入的理解对方的观点，从而反思自身观点的对错与不足；但若上升到人身攻击就会演变为无意义的争吵。

因此在 RIA 社群内讨论话题时，请尽量做到以下几点：

**讨论时请勿：**

- 情绪化泼冷水（“你这能有啥用？”）
- 消极沟通（“好好好；行行行你开心就好”）
- 对人攻击（“你怎么总想出这种破东西？”）
- 讲争辩性讨论归类为“杠”（“能别杠我了不？”）
- 不吸收观点（“反正我就这么觉得。”）

**讨论时推荐：**

1. 明确罗列自己的观点，并予以充分的论证；
2. 对奇怪的观点要求进一步说明，并属于较为柔和的表达（“你说的这些我不太清楚，可以说的再详细些么？”）
3. 遇到攻击时直接说明自己的心情（“你这个让我感觉到不愉快，...”）
4. 直接指出对方说服力不足（“你这并不能说服其他人，....”）
5. 反思自己是在坚持情绪还是在坚持道理。

**总结：快乐的讨论，而不是愤怒的攻讦：这是讨论的准则。**

最后，是如果违反了上述红线的惩罚：

前期较轻情节的将会给予提醒与警告，如果还继续违反的话就有可能被关紧闭：一段时间内不能进入RIA，登入将会被传送到专门的禁闭岛，最后，情节严重的将会被移除出白名单，并永远不会再次被RIA以及白名单共享的其他服务器接纳，请注意。
