欢迎虾仁来到 RIA（或称：日啊/莉娅），我是 RIA 玩家、这本手册的编者 TY！欢迎来到这一各种元素和谐共存，自由度极高的 MC JAVA 版服务器群组，开启一段新的旅程！RIA 下辖 5 个子服，按照开服顺序依次为：**零洲，奈落，后土，FST 服 Eden，FST 服 Laputa**，你可以任意挑选一个或多个游玩，关于这些子服的特性和区别，我会在第二章详细介绍。

你可以在服务器里做任何你想做的事！（前提是不会影响到他人或社区环境），关于你能做的和不能做的，我会在第五章详细介绍。

当然，你看到这本指南时，或许只是刚刚接触到这个服务器，所以我会在第一章带你做好所有必须的和可选的准备，同时，我会在每一章最后附上有关常见问题的 Q&A，帮助你解决问题。

如果你是还未获得资格的虾仁，你只需要了解前一到两章即可；如果你已经可以进入 RIA 了，那么我推荐你阅读全篇。

最后，未来我会隔一段时间（每季度或每半年，取决于服务器更新内容量）更新此指南，以保证其时效性。

现在，和我一起进入 RIA，享受这个世界吧！
